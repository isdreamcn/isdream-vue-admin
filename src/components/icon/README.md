# Icon

> 基于 [el-icon](https://element-plus.org/zh-CN/component/icon.html) + [iconfont](https://www.iconfont.cn/)

## API

### Icon 属性

| 属性名 | 说明                                            | 类型     | 预设值  |
| ------ | ----------------------------------------------- | -------- | ------- |
| size   | font-size                                       | `number` | inherit |
| color  | color                                           | `string` | --      |
| name   | `icon-` element-plus icons `iconfont-` iconfont | `string` | --      |

### A 插槽

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义默认内容 |
