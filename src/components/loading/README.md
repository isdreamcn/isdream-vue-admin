# Loading

## API

### Loading 属性

| 属性名  | 说明             | 类型      | 预设值 |
| ------- | ---------------- | --------- | ------ |
| loading | 是否显示 loading | `boolean` | false  |

### Loading 插槽

| 插槽名  | 说明                |
| ------- | ------------------- |
| default | 自定义 loading 内容 |
